# Ashwake Main Menu - 0.1.0

## Highlights
- Ashwake-themed main menu with cinematic background and custom button stack.
- Play Hub split for cleaner top-level navigation.
- First-run onboarding sequence with Guidance handoff.

## Fixes
- Added safe callback wrappers to prevent third-party API actions from crashing menus.
- Added sharp-rendering policy to avoid blur on Ashwake screens.

## Known Issues
- Remote changelog fetching is currently placeholder-only and falls back to local markdown.
- Texture-pack specific menu art is not yet included in this repository.

## Pack Changes
- Introduced configurable animation, particle, and performance profile toggles.
- Added Discord link placeholder and disable-state behavior.

## Mod Updates
- Added public API package for menu actions, guidance cards, and changelog entries.
